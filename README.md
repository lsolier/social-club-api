# Social Club Api

## Description

The purpose of this application is to create a social club system.


![context-diagram](https://github.com/lsolier/social-club-api/blob/main/assets/social-club-api.jpeg)

## Capabilities

- Create Members
- Create Clubs
- Member-Club Partnership

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
